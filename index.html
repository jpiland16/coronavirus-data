ABOUT THIS SITE

NOTE: This site has been confirmed to work on Google Chrome and Microsoft Edge (Chromium) - latest versions. Other browsers may not be supported.

This site displays information about COVID-19 cases by county and state level. Data is obtained from Johns Hopkins University on GitHub.

Access to this data is performed through Cross-Site Scripting made available through a demo at HerokuApp.

To get started, select a state and a county. Data will be shown for that county. Selecting another county will add that county to the graph. Counties can be removed at anytime using the list at the bottom of the page.

The y-axis can be set to be linear or logarithmic. The x-axis can be offset for each dataset showing the time since 5 cases were reported, or it can show all available data.

To view data by state, select "By State" rather than "By County." States can be added and removed in the same manner as counties.

Metrics available to analyze both states and counties include Total Cases, New Cases per day, New Cases per day (7-day rolling average), Percent Growth per day, and Percent Growth per day (7-day rolling average).

To view the top 20 states/counties based on the current metric, use the State Top 20 option to show the top 20 counties in the selected state, or use the US Top 20 option to show the top 20 states or counties in the United States. Note that data for counties with low numbers of cases may not be included.

To create a custom ranking, with the option to choose lowest (rather than highest) data, use the Custom Rank option.

The URL in your address bar is automatically updated with the options you have selected. To save your current layout, simply bookmark or favorite this page in your browser.
